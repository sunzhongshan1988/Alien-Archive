# Lucide Icons

Place shared Lucide SVGs for editor UI controls in this folder.

Suggested names for the current object/outliner buttons:

- `eye.svg`
- `eye-off.svg`
- `trash-2.svg`

Keep the upstream Lucide license/notice with any vendored Lucide SVGs. Lucide is distributed under the ISC License; some Feather-derived icons also carry the upstream Feather MIT notice.
