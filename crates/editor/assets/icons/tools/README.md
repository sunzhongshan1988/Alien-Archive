# Legacy Editor Tool Icons

Toolbar icons now use Lucide SVGs from `../lucide`. Keep this folder only as a historical fallback/reference for the earlier hand-drawn tool icons.

Deprecated file names:


- `select.svg`
- `brush.svg`
- `bucket.svg`
- `rectangle.svg`
- `erase.svg`
- `eyedropper.svg`
- `collision.svg`
- `zone.svg`
- `pan.svg`
- `zoom.svg`
- `fallback.svg`

Guidelines:

- Prefer a `24x24` viewBox.
- Use a single color via `currentColor`, `black`, or `#ffffff`; the editor will tint the icon.
- Keep shapes clear at small sizes.
- Do not include embedded bitmap images.
