# Editor tool icons

Put SVG toolbar icons in this folder. Use these exact file names:

- `select.svg`
- `brush.svg`
- `bucket.svg`
- `rectangle.svg`
- `erase.svg`
- `eyedropper.svg`
- `collision.svg`
- `zone.svg`
- `pan.svg`
- `zoom.svg`

Guidelines:

- Prefer a `24x24` viewBox.
- Use a single color via `currentColor`, `black`, or `#ffffff`; the editor will tint the icon.
- Keep shapes clear at small sizes.
- Do not include embedded bitmap images.
